# Design System: Hyper-Local Home
**Project ID:** 13218242358088433265

## 1. Visual Theme & Atmosphere
**Techno-Elegant Obsidian.** The project project presents a high-tech, deeply immersive "Matrix-lite" aesthetic. It balances the stark authority of a "Terminal" with the smooth, high-fidelity surfaces of modern glassmorphism. The density is medium-comfortable, utilizing generous whitespace and vibrant neon accents to guide the eye across deep obsidian backgrounds.

## 2. Color Palette & Roles
* **Obsidiana Profunda (#050E0A):** The primary canvas. Used for the main background to create depth and focus.
* **Lima Neón (#A3E635):** The primary action and brand color. Used for CTAs, highlighted text, and active states.
* **Verde Bosque Digital (#16A34A):** A secondary accent used for borders and subtle element separation to provide organic depth.
* **Neblina de Vidrio (rgba(134,239,172,0.1)):** Functional role for border outlines and dividers, providing a subtle "glowing" edge to glass components.
* **Texto Vaporoso (#EDFFF3):** High-contrast text color that ensures legibility against dark backgrounds while maintaining the greenish tint.

## 3. Typography Rules
* **Headers (Syne):** Heavy-weight (800+), tight letter-spacing (-0.03em) for a bold, technical stance. Used for H1, H2, and logo.
* **Body (Inter):** Clean, geometric sans-serif for readability. Used for all descriptive text, paragraphs, and nav links.
* **Data & Stats (DM Mono):** Fixed-width monospace to reinforce the "Terminal" and "Console" character of data displays.

## 4. Component Stylings
* **Buttons:** 8px Rounded corners. High-contrast Lima Neon backgrounds for primary actions with dark text (#000) for maximum punch.
* **Cards/Containers:** 12px Rounded corners. Backgrounds use `Glassmorphism` (blur + low-opacity overlay) with subtle `0.08` opacity green borders to create the illusion of glowing glass panels.
* **Inputs/Forms:** Dark, inset backgrounds (#081208) with 8px rounded corners and glowing borders on focus.

## 5. Layout Principles
* **Whitespace Strategy:** Generous top/bottom padding (100px+) between sections to allow elements to "breathe" in the dark space.
* **Grid Alignment:** Use of a 60px grid pattern in the background (Hero) to anchor the technical feeling of the layout.
* **Micro-motion:** Perpetual pulse animations for status badges and smooth fade-up transitions for section entry to ensure the UI feels "alive."
